# Net-Banking-Project
 Netbanking Web Application
Tech Stack: PHP, MySQL, HTML, CSS, JavaScript, XAMPP

A PHP-based Netbanking system simulating essential banking operations:

🔐 Login System: Secure session-based user authentication

💰 Account Dashboard: Displays real-time account balance

➖ Debit Feature: Allows users to perform debit transactions with input validation

💾 MySQL Database: Stores user credentials, balances, and transaction logs

🛡️ Security Measures: Prevents unauthorized access and SQL injection

🎨 User Interface: Clean, responsive frontend with HTML/CSS
